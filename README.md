# bugsnag-react-native

## Installation (dev mode)

1. Add `bugsnag-react-native` to your `package.json`:

   ```json
   {
     "dependencies": {
       "bugsnag-react-native": "../"
     }
   }
   ```

2. Run `npm install`

3. Run `react-native link`

4. Run your project with either `react-native run-android` or
    `react-native run-ios`
